# pothole-image-segmentation
本项目提出了一种基于YOLOv8的路面坑洞检测任务方法，实现了对坑洞的自动化识别和定位。

### 开发环境
windows 11
python 3.11
Ultralytics
以及各种库（详见代码）

### 数据集
来源于kaggle网站的[Pothole Image Segmentation Dataset](https://www.kaggle.com/datasets/farzadnekouei/pothole-image-segmentation-dataset)
请前往网站自行下载
